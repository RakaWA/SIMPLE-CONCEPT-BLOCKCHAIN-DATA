Requirements:
1. [Arangodb](https://www.arangodb.com/)
2. [NodeJS](https://nodejs.org/)

How to use:
1. Create your name database and collection in Arangodb
2. Edit file example-env.txt and rename file with ".env"
3. `npm i`
4. `npm start` or `node ./src`
